# uek335

## Installation

1. Clone this repo
2. Open it in a terminal
3. Type `npm i`

<br>

## To start the project

Type `npm start`

### Phone

Scan this QR code with Expo Go (Android) or the Camera app (iOS)

### Emulator

Type `a` to start for an Android device or `i` for an iOS device

### To stop the project

Type `Win + c`

### To reload the project

Type `r`
